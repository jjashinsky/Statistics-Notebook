table( )

Console Help Command: ?table

table( dataset$column )

table( dataset$column1, dataset$column2)

  • dataset is the name of your dataset while column is the name of one of the qualitative columns of that dataset. The table function then tallies up how many times each value of the quantitative variable occurs in that column.
  • column1 and column2 are two different columns of the dataset. They should both be quantitative. The table function tallies how many times each possible combination of column1 and column2 occur in the dataset.
